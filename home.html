I'm Tirso Peguero...

I'm one of those guys that likes problems. I like them because they have to be solved and the solving part can be really interesting. I've discovered over the last few years that my favorite problems are coding problems (I'm not currently taking any additional marriage counseling clients). I especially like those tricky problems that make you question the how or why of the project and force you to rethink your implementation.

I've been working with computers since before Windows 95 was released (does anyone else remember those 13 beige-colored 3.5" floppy disks?). I first started playing games on a Commodore 64 when I was a wee lad and soon graduated to piecing together computers with my dad - using DOS at first and then staying up late at night to install Windows. I'm also probably one of the few kids that went to web-dev summer camp in 5th grade... Fast-forward to college and I'm studying Military History and Russian Language while working tier 1 tech support for my school.

After working through several tech support positions I landed an entry level job with an advertising technology company that I rapidly grew into a product management role. After realizing that what I really wanted to do was build things I had no choice but to make the switch to engineering.

Some of my most recent projects are posted on the main page, but there's more on my github - go ahead and check out the links at the bottom of this page to see the rest of my online presence. If you've got some cool problems to work on and think I might be a fit, please don't hesitate to drop me a line.

Recent Projects

Cellō-Dev

Cellō-Dev is a project management web-app for Agile software development. Key features:

  • Server-side processing enables weekly backlog overflow.
  • Backbone frontend implements deeply nested views to enable flexible UI.
  • DOM alterations trigger events across Backbone views to ensure data changes are reflected both server-side and within Backbone framework.
  • Utilized global Backbone collection and filters to limit re-querying for data.

Giver

Giver is a clone of basic Rails functionality. Key features:

  • Controller and Router functionality to parse and direct HTTP requests.
  • Response parameter parsing allows for deeply nested form parameters.

Keeper

Keeper is a clone of basic ActiveRecord functionality. Key features:

  • AR associations written via Ruby meta-programming.

ASTEROIDS

A JS implementation of your favorite arcade classic! Featuring:

  • Uses trigonometry to calculate velocity vectors for all objects.
  • Displayed in browser using Canvas.
  • Your very own Death Star!
